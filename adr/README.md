IdeaWalker (IW)

ADR Directory README

This directory contains the official Architecture Decision Records (ADR)
for the IdeaWalker project.

  ---------------------------------------------------------------------
  ## Purpose
  ---------------------------------------------------------------------
  ## Directory Contents

  - ADR_INDEX.md → Catalog of all ADRs and their status -
  ADR_TEMPLATE.md → Template for new ADR creation - ADR-000 … ADR-XXX →
  Individual decision records
  ---------------------------------------------------------------------

Governance Model

1.  Any structural or epistemological change requires a new ADR.
2.  ADRs are immutable once Accepted.
3.  Superseded ADRs remain in the directory for traceability.
4.  This README must remain aligned with ADR_INDEX.md.

  ---------------------------------------------------------------------
  ## Integration Boundary
  ---------------------------------------------------------------------
  ## Workflow

  1. Create ADR-XXX using ADR_TEMPLATE.md. 2. Set Status → Proposed. 3.
  Review and approve. 4. Change Status → Accepted. 5. Update
  ADR_INDEX.md.
  ---------------------------------------------------------------------

End of Document
