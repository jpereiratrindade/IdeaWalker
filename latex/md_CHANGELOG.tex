\hypertarget{md_CHANGELOG_autotoc_md2}{}\doxysubsection{\texorpdfstring{Added}{Added}}\label{md_CHANGELOG_autotoc_md2}

\begin{DoxyItemize}
\item {\bfseries{Static Preview for External Files}}\+: Replaced continuous physics visualization with a pre-\/calculated, static rendering for Markdown/\+Mermaid previews. This ensures graphs are strictly organized and instantly stable upon loading.
\item {\bfseries{Enhanced Mermaid Parser}}\+:
\begin{DoxyItemize}
\item Added support for {\ttfamily mindmap} syntax with indentation-\/based hierarchy.
\item Added support for isolated node definitions (e.\+g., {\ttfamily A\mbox{[}Label\mbox{]}}) without requiring explicit links.
\item Improved support for node shape syntax like {\ttfamily ((circle))}.
\end{DoxyItemize}
\item {\bfseries{Text-\/\+Aware Layout}}\+: Graph physics now accounts for node text length to effectively prevent overlap.
\item {\bfseries{Radial Coloring}}\+: Static preview nodes are colored based on their angular position relative to the graph center.
\end{DoxyItemize}\hypertarget{md_CHANGELOG_autotoc_md3}{}\doxysubsection{\texorpdfstring{Fixed}{Fixed}}\label{md_CHANGELOG_autotoc_md3}

\begin{DoxyItemize}
\item {\bfseries{Neural Web Disappearance}}\+: Resolved a conflict where the main neural graph and the preview graph shared the same {\ttfamily Im\+Nodes} context. Implemented strict context isolation.
\item {\bfseries{Segmentation Fault on Startup}}\+: Fixed an initialization order issue where graph contexts were accessed before creation.
\item {\bfseries{Node Overlap}}\+: Nodes with long text no longer overlap in the preview.
\end{DoxyItemize}\hypertarget{md_CHANGELOG_autotoc_md4}{}\doxysubsection{\texorpdfstring{Changed}{Changed}}\label{md_CHANGELOG_autotoc_md4}
